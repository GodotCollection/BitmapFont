BitmapFont
==========

decode bitmap fonts, as 

	ASC12
	ASC16
	ASC48
	HZK12
	HZK14
	HZK16
	HZK16F
	HZK16S
	HZK24F
	HZK24H
	HZK24K
	HZK24S
	HZK24T
	HZK32
	HZK40S
	HZK40T
	HZK48
	HZK48S
	HZK48T

------

$ make 

to build

------

$ make clean

to cleanup

------

$ make font_name

ex. make ASC12 to get the pattern list text in UTF-8 of the ASC12 Font in output/

-----

$ make all

to all pattern list for all the fonts in output/
